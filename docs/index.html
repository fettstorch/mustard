What is Mustard?

Mustard lets you annotate any webpage with notes — like adding your "mustard" (from the German saying "seinen Senf dazu geben"). Save notes locally or publish them to your followers on Bluesky (or GitHub), who can comment on your published notes to start a conversation right where it's relevant.
Like a note someone shared? Repost it so your own followers can see it too — even if they don't follow the original author.

How to use

  1. Right-click anywhere on a webpage and select "Add Mustard" — or, when enabled in the options, hold Alt and click anywhere for a faster shortcut
  2. Write your note (up to 300 characters) — type @ to mention your Bluesky mutuals or GitHub follows; they get a heads-up in the popup's Mentions section
  3. Click the floppy disk to save locally (no account needed) or the blue arrow to publish (requires a login — connect Bluesky and/or GitHub from the popup)

Tips

  • Faster note creation — enable Alt+Click on the extension's options page to create a note by holding Alt and clicking anywhere on a page. It's off by default so it won't interfere with sites that already use Alt+Click.
  • Drag notes around if they're covering something you want to see
  • Toggle notes off via the extension popup — click the Mustard bottle icon in your toolbar
  • Minimize notes to small pills — toggle in the popup or options page, or press Alt+Shift+Z (⌥+Shift+Z on Mac); hover a minimized note to expand it
  • Add images by pasting an image link directly into your note
  • Repost a note from someone you follow so your own followers can see it too — hover a published note and click the repost icon. Reposters show up as small avatars stacked behind the original author's
  • Reply to a note — press the speech bubble icon on a published note to reply and start a conversation right there
  • Delete your own notes by clicking the bin icon

Markdown & gifs

  • Markdown supported — use **bold**, *italic*, [links](url), and more
  • Add gifs by typing / followed by your search — e.g. /wow opens the Giphy picker, then use arrow keys + enter to pick one

Ideas for your Mustard

  • Share your thoughts on something you just read
  • Provide additional context to a blog post
  • Fact-check a news article
  • Just slap a funny gif on something

Do not publish sensitive information

Never include passwords, API keys, personal addresses, or other private data in published notes. Published notes are stored on our server and visible to anyone who follows you.

Heads up

  • Content script on all pages: Mustard injects a content script on every page you visit to display notes. It does not read or collect any page data — it only checks if there are notes for the current URL.
  • Local notes are truly local: Notes saved locally never leave your browser. They're stored in your browser's extension storage.
  • Single page applications: Notes are tied to URLs. On single page applications where navigation doesn't always change the URL, notes may not appear on the correct view.
  • Early stage: Mustard is in early development. Expect rough edges 🙏🏻
